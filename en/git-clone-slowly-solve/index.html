д»ҘдёӢеҶ…е®№ж‘ҳеҪ•иҮӘ FastGithub.README

FastGithubдёәжҜҸеҸ°дёҚеҗҢзҡ„дё»жңәз”ҹжҲҗиҮӘйўҒеҸ‘CAиҜҒд№ҰпјҢдҝқеӯҳеңЁcacertж–Үд»¶еӨ№дёӢгҖӮе®ўжҲ·з«Ҝи®ҫеӨҮйңҖиҰҒе®үиЈ…е’Ңж— жқЎд»¶дҝЎд»»иҮӘйўҒеҸ‘зҡ„CAиҜҒд№ҰпјҢиҜ·дёҚиҰҒе°ҶиҜҒд№Ұз§Ғй’Ҙжі„йңІз»ҷд»–дәәпјҢд»Ҙе…ҚйҖ жҲҗжҚҹеӨұгҖӮ

гҖҠеӣҪйҷ…иҒ”зҪ‘жҡӮиЎҢи§„е®ҡгҖӢз¬¬е…ӯжқЎи§„е®ҡпјҡвҖңи®Ўз®—жңәдҝЎжҒҜзҪ‘з»ңзӣҙжҺҘиҝӣиЎҢеӣҪйҷ…иҒ”зҪ‘пјҢеҝ…йЎ»дҪҝз”ЁйӮ®з”өйғЁеӣҪе®¶е…¬з”Ёз”өдҝЎзҪ‘жҸҗдҫӣзҡ„еӣҪйҷ…еҮәе…ҘеҸЈдҝЎйҒ“гҖӮд»»дҪ•еҚ•дҪҚе’ҢдёӘдәәдёҚеҫ—иҮӘиЎҢе»әз«ӢжҲ–иҖ…дҪҝз”Ёе…¶д»–дҝЎйҒ“иҝӣиЎҢеӣҪйҷ…иҒ”зҪ‘гҖӮвҖқ FastGithubжң¬ең°д»ЈзҗҶдҪҝз”Ёзҡ„йғҪжҳҜвҖңе…¬з”Ёз”өдҝЎзҪ‘жҸҗдҫӣзҡ„еӣҪйҷ…еҮәе…ҘеҸЈдҝЎйҒ“вҖқпјҢд»ҺеӣҪеӨ–GithubжңҚеҠЎеҷЁеҲ°еӣҪеҶ…з”ЁжҲ·з”өи„‘дёҠFastGithubзЁӢеәҸзҡ„жөҒйҮҸпјҢдҪҝз”Ёзҡ„жҳҜжӯЈеёёжөҒйҮҸйҖҡйҒ“пјҢе…¶й—ҙжңӘеҜ№жөҒйҮҸиҝӣиЎҢд»»дҪ•йўқеӨ–еҠ еҜҶпјҲд»…жңүзҪ‘йЎөеҺҹжңүзҡ„TLSеҠ еҜҶпјҢеҢәеҲ«дәҺproxyзҡ„жөҒйҮҸеҠ еҜҶпјүпјҢиҖҢFastGithubиҺ·еҸ–еҲ°зҪ‘йЎөж•°жҚ®д№ӢеҗҺеҸ‘з”ҹзҡ„ж•ҙдёӘд»ЈзҗҶиҝҮзЁӢе®Ңе…ЁеңЁеӣҪеҶ…пјҢдёҚеҶҚйҖӮз”ЁеӣҪйҷ…дә’иҒ”зҪ‘зӣёе…ід№Ӣи§„е®ҡгҖӮ

Hosts дҝ®ж”№

дё»иҰҒжҳҜеҖҹеҠ© nslookup or dig иҺ·еҸ–еҹҹеҗҚеҜ№еә”зҡ„еҸҜи®ҝй—® ip, з„¶еҗҺеңЁ hosts ж–Үд»¶дёӯеҠ дёҠ ipвҖ“>domain зҡ„жҳ е°„пјҢеҲ·ж–° DNS зј“еӯҳдҫҝеҸҜгҖӮ

hosts ж–Үд»¶жҳҜ Linux зі»з»ҹдёӯдёҖдёӘиҙҹиҙЈ IP ең°еқҖдёҺеҹҹеҗҚеҝ«йҖҹи§Јжһҗзҡ„ж–Үд»¶,д»Ҙ ASCII ж јејҸдҝқеӯҳеңЁ /etc зӣ®еҪ•дёӢ,ж–Үд»¶еҗҚдёә hosts (дёҚеҗҢзҡ„ Linux зүҲжң¬,иҝҷдёӘй…ҚзҪ®ж–Үд»¶д№ҹеҸҜиғҪдёҚеҗҢ.жҜ”еҰӮ Debian зҡ„еҜ№еә”ж–Үд»¶жҳҜ /etc/hostname) hosts ж–Үд»¶еҢ…еҗ«дәҶ IP ең°еқҖе’Ңдё»жңәеҗҚд№Ӣй—ҙзҡ„жҳ е°„,иҝҳеҢ…жӢ¬дё»жңәеҗҚзҡ„еҲ«еҗҚ.еңЁжІЎжңүеҹҹеҗҚжңҚеҠЎеҷЁзҡ„жғ…еҶөдёӢ,зі»з»ҹдёҠзҡ„жүҖжңүзҪ‘з»ңзЁӢеәҸйғҪйҖҡиҝҮжҹҘиҜўиҜҘж–Үд»¶жқҘи§ЈжһҗеҜ№еә”дәҺжҹҗдёӘдё»жңәеҗҚзҡ„ IP ең°еқҖ,еҗҰеҲҷе°ұйңҖиҰҒдҪҝз”Ё DNS жңҚеҠЎзЁӢеәҸжқҘи§ЈеҶі.йҖҡеёёеҸҜд»Ҙе°Ҷеёёз”Ёзҡ„еҹҹеҗҚе’Ң IP ең°еқҖжҳ е°„еҠ е…ҘеҲ° hosts ж–Үд»¶дёӯ,е®һзҺ°еҝ«йҖҹж–№дҫҝзҡ„и®ҝй—®.
